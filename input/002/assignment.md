# JSON-andmete kuvamine

## Ülesande kirjeldus
Koosta rakendus, mis loeb JSON-failist andmed ja kuvab need veebilehel tabelina. Rakendus peab võimaldama andmeid filtreerida ja sorteerida.

## Nõuded
1. Andmed tuleb lugeda data.json failist.
2. Andmed kuvatakse HTML-tabelina.
3. Kasutaja saab andmeid filtreerida nime või kategooria järgi.
4. Kasutaja saab andmeid sorteerida veeru järgi.
5. Kui JSON-faili ei õnnestu lugeda, kuvatakse veateade.
6. Leht peab olema responsive.

## Hindamiskriteeriumid
- Andmete korrektne lugemine ja kuvamine (30%)
- Filtreerimise ja sorteerimise funktsionaalsus (30%)
- Veakäsitlus (20%)
- Responsive disain (10%)
- Koodi loetavus (10%)
