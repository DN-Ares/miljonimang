# JavaScripti kalkulaator

## Ülesande kirjeldus
Koosta lihtne veebikalkulaator, mis võimaldab kasutajal teha põhilisi aritmeetilisi tehteid: liitmine, lahutamine, korrutamine ja jagamine.

## Nõuded
1. Kalkulaatoril peab olema kasutajaliides HTML ja CSS abil.
2. JavaScript peab teostama arvutused.
3. Kasutaja saab sisestada kaks arvu ja valida tehte.
4. Tulemus kuvatakse lehel.
5. Vigase sisendi korral (nt tühi väli või tekst) kuvatakse veateade.
6. Nulliga jagamise korral kuvatakse veateade.

## Hindamiskriteeriumid
- Loetav ja hästi struktureeritud kood (30%)
- Õige funktsionaalsus (40%)
- Veakäsitlus (20%)
- Väljanägemine ja kasutajakogemus (10%)
